浅谈正向代理和反向代理

2018-09-10

正向代理

以前在网上看资料一直很迷惑什么是正向代理，看文章都是长篇大论，讲得云里雾里。
现在先看一下下面的正向代理的图


client就是我们的自己的电脑，当我们电脑像去访问一些科学网站会被墙直接拦下来，但是可以通过代理去访问某些网站，这种就是正向代理，我们是通过主动的形式访问代理服务器，这个代理我们是知情的。

反向代理

现在先看一下下面的反向代理的图


client就是我们的自己的电脑,当我们访问一些网站，我们访问的服务器有可能并没有我们所需的内容，而是从别的服务器拿来的，我们直接访问的服务器是一个代理服务器，通过这个代理服务器再去访问真正的服务器，最终返回我们所需要的网页或者数据，这个代理并不是我们自己主动去代理的，而是被动的，我们不知情的，这个就是反向代理。
