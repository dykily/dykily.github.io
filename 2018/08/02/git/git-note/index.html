git基本操作

git config --list 
显示当前git配置

git config --global user.name "fgq" 
设置提交仓库的用户名信息

git config --global user.email "250684079@qq.com"
设置提交仓库的邮箱信息

git init
初始化仓库

git clone
克隆操作

git add [file1] [file2]
添加文件到暂存区

git rm [file1] [file2]
删除工作区文件，并将这次删除放入暂存区

git mv [ofilename] [newfilename]
改文件名，并把这个名字放入暂存区

git commit -m [message]
提交暂存区的文件到仓库

git commit -a -m [message]
直接从工作区的文件提交到仓库
前提该文件已经在仓库有历史版本（就是之前已经从暂存区提交到仓库）

git status
查看变更信息

git log （git log --oneline）
查看当前分支的历史版本

git remote add [shortname] [url]
增加远程仓库，并可以命一个简短的别名

git push [remote/shortname] [branch]
将本地的仓库推送到远程仓库

git pull [remote/shortname] [branch]
把远程仓库的提交同步到本地仓库

git clone [url/ssh]
把远程仓库拉到本地来，这就可以自动和远程仓库关联了

vim基本操作

j 往上走一行 k 往下走一行
gg 最顶端 G 最底端 空格翻一页
/ 搜索 N（大写N） 上一个匹配 n 下一个匹配
