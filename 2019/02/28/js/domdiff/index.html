介绍

diff算法是进行虚拟节点Element的对比，并返回一个pathchs对象，用来存储两个节点不同的地方，最后用pathchs记录的消息去局部更新Dom。

两个树如果完全比较的话需要时间复杂度为O(n^3)，如果对O(n^3)不太清楚的话建议去网上搜索资料。而在Diff算法中因为考虑效率的问题，只会对同层级元素比较，时间复杂度则为O(n)，说白了就是深度遍历，并比较同层级的节点。
